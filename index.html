yabox.js by bebraw

Yet another lightbox clone. Very light one this time.

License

MIT

Dependencies

jQuery (1.something)

Installation

Copy jquery.yabox.js to some appropriate place, link to it at your HTML, use 
$(selector).yabox(opts);
Check out opts and markup at source.

Download

You can also clone the project with Git by running: 

$ git clone git://github.com/bebraw/yabox.js

Get the source code on GitHub : bebraw/yabox.js
